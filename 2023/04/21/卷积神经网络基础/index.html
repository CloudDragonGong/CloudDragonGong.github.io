卷积神经网络基础

本片博客对应着吴恩达教学视频的卷积神经网络的章节

1. Computer Vision

计算机视觉的目标任务

  • 物体分类
  • 神经风格迁移
  • 目标分割

神经风格迁移（Neural Style Transfer）：它是一种用于将两个不同图像的风格进行合成的技术。这个任务的目标是训练一个模型来将一张输入图像的内容与另一张参考图像的风格进行合成，生成一张新的图像，使其看起来既保留了原始图像的内容，又采用了参考图像的风格。这种技术可以用于图像编辑、图像生成等应用领域。

传统神经网络的问题就是由于一个图片直接放入神经网络参数太多，导致神经节点的数量太多，权重的数量庞大，没那么多算力，卷积可以减少计算量

2. Edge Detection Example

边缘有两种最常见的边缘

  • 垂直的边缘
  • 水平的边缘

提取边缘的特征（比如我是想提取垂直的还是想提取水平的）方式就要看卷积核的值取得怎么样，下面是提取垂直的特征，可以思考一下为什么这样设值是可以提取到垂直特征的，怎么样设可以提取到水平特征。

3. More Edge Detection

假设要提取垂直的边缘特征

有两种渐变会产生边缘效果

  • 从亮变暗
  • 从暗变亮

可以发现这个效果经过卷积核之后并没有发生改变，也就是特征提取出来了

但是我们不仅仅是提取简单的水平和垂直

我们要提取很多别的特征

这样的话就要设计算子，其他的算子

Sobel filter和Scharr filter

在深度学习中，卷积核是通过cnn训练学习得到的，cnn就是求出卷积核的数值，这样就可以针对性得更好的提取特征

4. Padding

俗称补零操作

上面发生卷积后图片的尺寸减小了，并且在进行卷积操作的时候，原始图片的边上那些像素点参与卷积运算的很少

为了让边上的像素点能够参与卷积运算，进行补零操作

没有padding操作，$p = 0$,称为“Valid convolutions”；

有padding操作，$p=\frac{f-1}{2}$，我们称之为“Same convolutions”。
