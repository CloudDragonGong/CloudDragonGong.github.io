MouseListener 接口

MouseListener 接口是 Java AWT/Swing 框架中的一个接口，用于处理鼠标事件。该接口中包含以下 5 个方法，可以在实现接口的类中进行重写：

  1. void mouseClicked(MouseEvent e)：在用户单击鼠标按钮时调用。该方法会接收一个 MouseEvent 参数，该参数包含了有关鼠标事件的详细信息。
  2. void mousePressed(MouseEvent e)：在用户按下鼠标按钮时调用。该方法会接收一个 MouseEvent 参数，该参数包含了有关鼠标事件的详细信息。
  3. void mouseReleased(MouseEvent e)：在用户释放鼠标按钮时调用。该方法会接收一个 MouseEvent 参数，该参数包含了有关鼠标事件的详细信息。
  4. void mouseEntered(MouseEvent e)：当鼠标进入组件的范围时调用。该方法会接收一个 MouseEvent 参数，该参数包含了有关鼠标事件的详细信息。
  5. void mouseExited(MouseEvent e)：当鼠标离开组件的范围时调用。该方法会接收一个 MouseEvent 参数，该参数包含了有关鼠标事件的详细信息。

需要注意的是，重写这些方法时，可以根据实际需求选择是否使用方法中的 MouseEvent 参数。例如，如果只需要在鼠标单击时执行某些操作，则可以只重写 mouseClicked 方法，并忽略其他方法。
